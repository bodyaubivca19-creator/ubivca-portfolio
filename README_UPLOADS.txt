Что реализовано:
- Добавление и редактирование проектов без правки кода
- Поддержка файлов любого формата через admin.html
- Публичный просмотр проектов для гостей
- Вход в админ-панель только для администратора
- Смена пароля администратора в разделе «Настройки»
- Переход в редактирование проекта прямо из открытого кейса
- Хранение проектов и файлов в IndexedDB браузера

Доступ в админку по умолчанию:
- логин: admin
- пароль: ubivca2026

После первого входа лучше сразу сменить пароль в настройках.

Важно:
Это статический сайт без сервера.
Поэтому пароль и проекты хранятся в браузере, а не на удалённом сервере.
Это хороший вариант для локального или простого хостинга, но не полноценная серверная безопасность.
Если нужен общий доступ с любого устройства, единая база проектов и настоящая авторизация, нужно подключить backend или CMS.
